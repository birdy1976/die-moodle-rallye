Die Internet-Rallye

Die Internet-Rallye ist eine Methode des Interneteinsatzes im Unterricht. Dabei bekommen die [Teilnehmenden] eine gewisse Anzahl von Fragen oder bestimmte Arbeitsaufträge (Bilder herunterladen, Informationen herausschreiben usw.), die sie mit Hilfe von bereits angegebenen Webadressen beantworten und lösen müssen. » Quelle...

Das Ziel

Gib so schnell wie möglich die Antworten auf die fünf Fragen. Die Lösung ersetzt dabei jeweils das letzten Wort der URL der aktuellen Seite. Wenn also z.B. die Seite http://die-agentur.ch/die-moodle-rallye/start/ und die Lösung mooodle lautet, dann muss in der URL start durch moodle ersetzt werden. Alles klar? Also los! » Start...
